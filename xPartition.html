xPartition

Usage

xPartition(pred, coll)


xPartition_(pred, ...)

Arguments

pred
a predicate. The predicate function with which to partition the input collection.
coll
a collection. The collection to partition into two collections.
...
see above.

xPartition

Value

Returns two lists; a list of elements in a collection for which a predicate returns true, and a list of elements in a collection for which a predicate returns false or na.

Description

Divide elements in a collection into two collections based on a predicate function.

Type Signature

(any -> <logical>) -> |any| -> [[any]]

Corner Cases

If coll is empty the empty list is returned. If all the elements return only true/only false, then one of two sublists will be the empty list.

Examples


# 1. Partitioning

mixed <- xShuffle(c(letters, LETTERS))

# -- is the letter a lower-case letter?
xPartition(
	xFix_(xIsMember, coll = letters),
	mixed)
