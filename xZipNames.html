xZipNames

Usage

xZipNames(colls)


xZipNames...(...)

Arguments

colls
a list or pairlist of two-element lists or pairlists, with the first element being a string and the second element being an arbitrary value. The string part of each sublist is used as the name in the output list, and the remaining part is used as the value associated with that key.
...
see above.

xZipNames

Value

A named list.

Description

Convert a collection of name, value pairs into a named collection.

Corner Cases

Returns the empty list if coll is length-zero.

Examples


# 1. turn a csv of country names and median ages into
#    a named list of countires, with values being the median age.

csv <- "Monaco,48.9,Japan,44.6,Italy,44.3,Germany,43.7,Jersey,43.4"

x_(csv) $ xExplode(',') $ xChunk(2) $ x_ZipNames()

$Japan

$Italy
