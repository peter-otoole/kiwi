xCombos

Usage

xCombos(num, coll)


xCombos...(num, ...)

Arguments

num
a nonnegative whole number. The number of elements to choose from coll.
coll
a collection. The collection to draw elements from.
...
see above.

Value

A list of lists, with each list containing num elements.

Description

Enumerate all ways of choosing several elements from a collection.

Details

xCombos enumerates all ways of choosing num distinct elements from a larger collection, where order doesn't matter.

The number of ways of choosing num elements from a collection is given by the base R function choose(num, length(coll)).

Corner Cases

Returns the empty list if num is zero.
