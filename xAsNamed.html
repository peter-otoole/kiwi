xAsNamed

Usage

xAsNamed(strs, coll)

Arguments

strs
a collection of strings. The names to add to the input collection.
coll
a collection. The collection to add names to.

xAsNamed

Value

A named list.

Description

Add names to a collection.

Details

xAsNamed is similar to the in-place assignment function names<-, except that it is not an in-place assignment function.

Corner Cases

Returns the empty list if coll is length-zero. Both duplicated and length-zero names are allowed.
