xAsInteger

Value

An integer vector.

Description

Convert a collection to a integer vector.

Details

xAsInteger converts a list, pairlist or vector of length-one integers to a integer vector. It does not attempt to convert non-integer collections to integer vectors.

Corner Cases

Double vectors are not coerced to integer vectors: doubles can be decimal or infinite numbers, which would be lost upon coersion.
