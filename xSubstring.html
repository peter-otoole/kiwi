xSubstring

Usage

xSubstring(str, nums)


xSubstring...(str, ...)

Arguments

str
a string. The string to subset.
nums
a collection of whole numbers. The indices to select characters with. Negative, positive and zero indices are allowed, but normal R index semantics dictate how they can intermix.
...
see above.

xSubstring

Value

A character vector.

Description

Subset a string using normal R vector indexing.

Corner Cases

Returns the empty list if coll is length-zero. If any number in nums is larger than the length of coll an error is thrown.
