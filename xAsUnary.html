xAsUnary

Usage

xAsUnary(fn)

Arguments

fn
an arbitrary function. The function to be converted to a function that takes a single collection.

xAsUnary

Value

A unary function of coll, that applies its arguments to its underlying function.

Description

Convert any function to a unary function.

Details

xAsUnary takes a function and returns a function that has one argument. The first element of the argument is passed to the first parametre of the underlying function, the second element to the second parametre, and so on.

Examples


# 1. xAsUnary can be used to adapt higher-order
#    functions to make them work with non-unary functions.

xSelect...(
    xAsUnary(
        (a : b) := {
            a > 0 && b >= 0
        }
    ),
    list(+1, +1),
    list(+0, +1),
    list(-1, 1),
    list(-1, -1)
)
