xRead

Usage

xRead(str)

Arguments

str
a length-one character vector. The path to import files from.

xRead

Value

A length-one character vector.

Description

Read the contents of a file as a single string.

Type Signature

|character| -> <character>

Corner Cases

xRead ignores missing terminal newlines. If str is is length-zero the empty character-vector is returned.

Examples


# 1. Read the first line of kiwi's description.

x_(system.file(package = 'kiwi', 'DESCRIPTION')) $
xRead() $ xToLines() $ x_Take(1)
