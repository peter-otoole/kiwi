xTabulate

Usage

xTabulate(coll)


xTabulate...(...)

Arguments

coll
a collection. The values to find the frequency of.
...
see above.

xTabulate

Value

A list.

Description

Tabulate a collection into pairs of value:frequency lists.

Details

xTabulate is superficially similar to the base function table: given a collection that may or may not contain duplicates, it calculates the frequencies of each unique element.

xTabulate(c('y', 'n', 'y', 'y', 'n'))

list(list("y", 3), list("n", 2))

The result of the tabulation is unsorted for efficiency; if sorting is required xSortBy can be used.

Corner Cases

If coll is length-zero the empty list is returned.

# a subset of the 1999 Clinton impeachment vote.
# each row of the data represents a candidate;
# the columns the party, perjury vote and obstruction vote respectively.

	R	Guilty		Guilty
	D	NotGuilty	NotGuilty
	R	Guilty		Guilty
	R	Guilty		Guilty
	D	NotGuilty	NotGuilty
	D	NotGuilty	NotGuilty

	D	NotGuilty	NotGuilty
	D	NotGuilty	NotGuilty
	R	Guilty		Guilty
	D	NotGuilty	NotGuilty
	D	NotGuilty	NotGuilty
	R	Guilty		Guilty
	D	NotGuilty	NotGuilty
	R	Guilty		Guilty
	R	Guilty		Guilty
	D	NotGuilty	NotGuilty
	R	Guilty		Guilty
	R	NotGuilty	NotGuilty
	D	NotGuilty	NotGuilty
	R	Guilty		Guilty
	R	NotGuilty	NotGuilty
	D	NotGuilty	NotGuilty
	R	Guilty		Guilty
	R	Guilty		Guilty
	D	NotGuilty	NotGuilty"

clinton_vote <-
	x_(raw_clinton_vote) $ xToLines() $ xMap(xToWords) $
	xZip() $ xAddKeys(c('party', 'perjury', 'obstruction'))

# 1.
# Tabulate the overall results for the perjury vote

clinton_vote $ xAtKey('perjury') $ x_Tabulate()

# 3.
# An advanced example;
# get the overall vote within each party.

clinton_vote $ xZip () $ xGroupBy(xFirstOf) $ # -- group by party
xAtCol(2) $ # -- select the groups
xMap(party := {
	# get the perjury and obstruction votes by party.

	list(
		perjury =
			x_(party) $ xAtCol(2) $ xTabulate() $ x_ZipKeys(),
		obstruction =
			x_(party) $ xAtCol(3) $ xTabulate() $ x_ZipKeys()
	)

}) $
x_AddKeys(c('Republican', 'Democrat'))
