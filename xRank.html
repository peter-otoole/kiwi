xRank

Usage

xRank(nums)


xRank...(...)

Arguments

nums
a vector of numbers. The numbers to rank in order of size.
...
see above.

Value

A vector of numbers.

Description

Rank a collection of numbers from largest to smallest.

Corner Cases

If nums is empty then integer(0) is returned. Tied values are ranked in order of appearance; the first occurrence is ranked lower than the second occurrence.
